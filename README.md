# 🧠 AI Emotion Engine

> **The Future of Human-Centric Sentiment Intelligence.**

![Design](https://img.shields.io/badge/UI-Futuristic-blueviolet)
![Engine](https://img.shields.io/badge/AI-TensorFlow-orange)
![License](https://img.shields.io/badge/License-MIT-green)

A state-of-the-art **Facial Emotion Detection** system featuring a premium Glassmorphism UI, real-time vision analytics, and localized voice feedback.

---

## ✨ Key Features

- **🛡️ Real-time Detection**: Millisecond-accurate facial tracking via OpenCV.
- **🧠 Deep Learning Engine**: High-accuracy CNN trained on the FER2013 dataset.
- **🔊 Voice Intelligence**: Hindi voice assistant announces emotional states.
- **✨ Future UI**: Stunning Glassmorphism dashboard built with Streamlit & Custom CSS.
- **📉 Analytics Dashboard**: Interactive confidence charts for every prediction.
- **🧩 Smoothing AI**: Advanced sliding-window logic to eliminate prediction flickering.

---

## 🚀 Quick Start

### 1. Installation
```bash
git clone https://github.com/alokyadav9045/Emotion-ai.git
cd Emotion-ai
pip install -r requirements.txt
```

### 2. Execution
```bash
streamlit run app.py
```
